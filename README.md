# ECON 3916 Final Project — Ames Housing Price Predictor

## Overview
Predicting home sale prices in Ames, Iowa using physical attributes, condition, and location.

## Setup
```bash
pip install -r requirements.txt
```

## Data
The dataset is included in the `data/` folder (`ames_housing.csv`). Source: De Cock, D. (2011). Ames, Iowa: Alternative to the Boston Housing Data. Journal of Statistics Education, 19(3).

## Reproducing the Analysis
1. Open and run `3916-final-project-starter (1).ipynb` sequentially
2. Run `python save_model.py` to save the trained model

## Streamlit App
```bash
streamlit run app.py
```
